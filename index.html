ZED Champions Import Setup

Enter your Bearer Token to import real horse data from ZED Champions

How to get your Bearer Token:

  1. Open ZED Champions in your browser
  2. Press F12 to open Developer Tools
  3. Go to Network tab
  4. Refresh the page
  5. Look for requests with "Authorization: Bearer" header
  6. Copy the token value (without "Bearer " prefix)
